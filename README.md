# DPPForge

Digital Product Passport (DPP) SaaS platform focused on product verification and anti-counterfeit protection.

## Features

- QR verification system
- Fraud detection infrastructure
- Real-time analytics dashboard
- Product scan tracking
- Authentication & authorization
- Email notification system
- Admin management panel
- Suspicious activity detection

## Technologies

- Next.js
- TypeScript
- Supabase
- PostgreSQL
- Tailwind CSS
- Vercel

## Purpose

DPPForge helps brands protect their products against counterfeiting by providing secure digital verification and analytics systems.

## Status

Active development and deployment process continues.
